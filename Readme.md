## 二、 开发环境及工具

本工程是一个火柴人战争的游戏。

本工程基于cocos2d-x引擎。

整体架构参考了Model-View-ViewModel（MVVM）实现。

## 三、 小组内分工

第一轮迭代由胡啸川完成部分Model层，

第二轮迭代由胡啸川完成ViewModel层以及游戏主界面的View层相关工作，。

第三轮迭代由胡啸川完成更多的Model层，完善了View层的相关功能，。

三轮迭代完成，游戏完成。
